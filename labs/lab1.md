# Lab 1: Half Adder and 4-to-1 Multiplexer

## Half Adder

### Procedure

```
$ cd C:/eda/half adder
$ ghdl -a ha.vhdl
$ ghdl -a ha_tb.vhdl
$ ghdl -e ha_tb
$ ghdl -r ha_tb --vcd=ha.vcd
$ gtkwave ha.vcd
```
